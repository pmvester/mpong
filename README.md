# mpong
Simple pong game.
## Installing
```bash
git clone https://github.com/pmvester/mpong
cd mpong
npm ci
```
## Playing
### Start servers
```bash
node pong-server.js
```
and in another terminal window
```bash
node express-server.cjs
```
### Play game
Open two (2) web browsers pointing to:
http://localhost:8080
Move paddles by pressing 'w' (up) and 's' (down).
